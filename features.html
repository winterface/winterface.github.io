Key Features

The new re-imagined Anixter Power Solutions site offers a modern but accessible website that works on any device. With powerful (but easy to use) tools, Anixter Power Solutions customers will have one site that will provide them with access to everything they need to be successful. Read below for a list of key features that make the new Anixter Power Solutions website stand above the rest.

General

  • Fully responsive. Works on iPhones, Android Phones, Tablets, Laptops, Desktops, and even large scale displays.
  • Optimized to load quickly allowing users on any speed connection to use the website smoothly.
  • Full support for Internet Explorer, Firefox, Chrome, Opera. Windows and Mac support as well.
  • Streamlined content enabling users to access what they need quickly and efficiently.
  • Strong call to actions to get users to sign up for The Current, Storefront, and methods of contact.
  • Built upon a modern framework that will allow a high degree of future proofing and allow for easy modifications and additions in the future

Location Finder

  • Search for Anixter Power Solutions Locations by Address or Zip Code.
  • Modify search radius options to find nearby locations
  • Generate turn-by-turn directions for any location.
  • Ability to see weather conditions for each particular location; including current conditions, upcoming hour, 24 hours, and any watches/warnings that may affect safety and operations in the area.
  • Sidebar with locations listed in A-Z order for users that don't want to use the map.

Accessibilty

  • Adheres to Accessbility guidelines for users with disabilities (US Section 508)
  • Tool to translate language on website into French, Spanish, German, or Italian.
  • Fully optimized SEO with meta data,alt tags, title tags and keyword optimization.
  • "Thumb Friendly" mobile interface that allows users to use the site one handed.

New and Existing Users

  • Interactive tutorial that guides users around the website and explains the core features.
  • Mobile friendly password field for login. Users can toggle their password to see if they have made mistyped or are typing the wrong password.
  • Easy and and clear access for registering as a new user, resetting a password, and multiple avenues of contacting support.
  • A seamless login experience: Users will no longer be forced to juggle new tabs or new windows upon logging in. Logged in user content appears dynamically in the user's current page and offers the user a quicker, easier method of accessing their single sign-on applications.

Branding/Marketing

  • Banners between each site section displaying an Anixter slogan.
  • Strong Call-to-Actions for Storefront
  • Links back to Anixter.com
  • Logo, Color Scheme, and other UI elements that match the overall look and feel of the Anixter brand.
